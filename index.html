What is Lodash?

JavaScript in its original form can (at times) be a bit tough for a person to know what to code in order to accomplish the goal that person has in mind. It can also be tedious at times to write out all of your commonly used functions, loops, and such over and over again.

Enter, Lodash. Lodash is what's referred to as a "reference library" for the JavaScript programming language. It's a collection of pre-written code that allows coders to work faster, smarter, and with consistency. Here are some this particular library's capabilities:
